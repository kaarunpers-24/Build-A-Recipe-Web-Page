Building A Recipe Web Page
A simple and responsive recipe webpage for French Bread Toast, created using HTML as part of the FreeCodeCamp Responsive Web Design lab.

## About the Project
The webpage presents a French Bread Toast recipe with:
- A clear title and description
- An ingredients list
- Step-by-step cooking instructions
- An image of the final dish

This project focuses on practicing **basic HTML structure and semantic elements**.

# Built With
HTML5

##  Learning Outcomes
- Using semantic HTML elements
- Working with headings, paragraphs, lists, and images
- Structuring a clean and readable webpage
- Following FreeCodeCamp project requirements

You can view the project by opening the `index.html` file in a browser
